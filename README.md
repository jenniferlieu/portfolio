My portfolio website to showcase my projects.

Built with Next.js, Tailwind CSS, and Contentlayer. Deployed on Vercel.


## Getting Started

1. Clone this repo
2. Install dependencies
   ```bash
   npm install
   ```
- Run dev server
    ```bash
    npm run dev
    ```
Open [http://localhost:3000](http://localhost:3000).

- Build and serve
   ```bash
   npm run build
   npm run start
   ```
